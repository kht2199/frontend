# Feature List

## Campus3D (3D 캠퍼스 뷰)

### 렌더링
- GLB 모델 로드 (Draco 압축, 진행률 표시)
- ACES Filmic 톤매핑, PCF 소프트 그림자
- @react-three/fiber 기반 렌더링

### 조명 / 환경
- 시간 모드 선택: 아침(10시) / 밤(0시) / 실시간(현재 시각)
- 시간대별 하늘 색상 변화 (새벽 노을 → 낮 하늘 → 밤하늘)
- 태양·달 메시: 반원 궤도 애니메이션, 글로우 후광
- 방향광이 태양 위치를 따라 이동

### 건물 인터랙션
- m14 / m16 건물 호버 → 커서 pointer 변경
- m14 / m16 건물 클릭 → 마우스 우측에 정보 팝업 표시 (건물명, 설명, 위치)
- m14 / m16 상시 라벨: 건물 위에 이름·위치 표시 (3D→2D 투영)
- 빈 공간(오브젝트 외부) 마우스 좌클릭으로 팝업 닫기

### 건물 효과
- 경고등(wrn 재질) 점멸 애니메이션
- 경고등 위치마다 연기 파티클 (드리프트 + 리셋)
- 창문(wm 재질): 주간 하늘빛 유리 ↔ 야간 주황 발광 자동 전환
- 경고 건물 단일 선택 → 해당 건물 전체 빨간 점멸 (내부 데이터는 배열 유지)

### 카메라 / 컨트롤
- OrbitControls (드래그·줌·패닝)
- 수평·수직 회전각 5° 단위 스냅
- 카메라 XYZ 위치 / 거리(Dist) / 타겟 XYZ 수동 입력
- 건물 선택 드롭다운으로 해당 건물 포커스
- Reset 버튼으로 초기 카메라 위치 복원

### 미니맵
- 우측 하단 탑뷰 미니맵 (OrthographicCamera scissor 렌더)
- 카메라 위치 점 + 시야각 부채꼴 오버레이 (2D canvas)
- m14 / m16 건물 풋프린트 폴리곤 마커
- 클릭 또는 드래그로 카메라 타겟 이동

## AlarmPanel

### 탭 구성
- **Today's Alarm**: 기존 알람 목록
- **Trend**: 3개 ECharts 바차트

### Trend 차트
| 차트 | 종류 | 비고 |
|------|------|------|
| Canceled Queue | 바차트 | 값 라벨 표시 |
| OHT Usage | 바차트 | 값 라벨 표시, y축 % 없음 |
| Queue Saving | 스택 바 + 라인 | Saved(초록) + Potential(연초록) 스택, Target(빨강 점선) + Baseline(파랑 점선) 동적 라인 |

- x축: 문자열 레이블 (Bay-A ~ Bay-H)
- 데이터: `/public/trend-*.json` 파일에서 fetch
- 탭 전환 시 차트 resize 자동 실행
